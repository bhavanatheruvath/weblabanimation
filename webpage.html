What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It defines how elements should be rendered on various media such as screens, paper, or speech 1 . CSS is a core language of the open web and is standardized across web browsers according to W3C specifications 1 .
